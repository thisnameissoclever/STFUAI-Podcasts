# STFUAI Podcasts

[STFUAI Podcasts](https://stfuai.com) is a podcast player and manager that automatically skips ads and other skippable segments such as intros, outros, closing credits, and self-promotion segments. It was built by [Tim Woodruff](https://li.snc.guru): [developer](https://snprotips.com), [author](https://books.snc.guru), and disliker of intrusive and annoying ads.

STFUAI Podcasts is short for "**S**kip **T**he **F**\*cking **U**nwanted **A**ds **I**n Podcasts".
